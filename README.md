## ✨ Overview

A collection of small Go projects and exercises used for learning the language and its standard library.

## 🚀 Getting Started

### Prerequisites

- [ ] Go 1.22+

### Installation

```bash
git clone https://github.com/ZakeCode/go-adventures.git
cd go-adventures
```

### Usage

```bash
go run .
```
